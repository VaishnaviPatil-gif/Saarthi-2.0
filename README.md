# Saarthi – Local AI Chat Backend

This adds a minimal Express server that powers the AI chatbot in `Saarthi.html`.

## Prerequisites
- Node.js 18+ (recommended)

## Setup (Windows PowerShell)

```powershell
# 1) Go to the server folder
cd "c:\Users\vaish\OneDrive\Desktop\saarthi_new\server"

# 2) Install dependencies
npm install

# 3) Create your .env from the example
Copy-Item .env.example .env

# 4) Edit .env and add your OpenAI API key
#    OPENAI_API_KEY=sk-...

# 5) Run the server
npm run dev
```

By default the API runs at http://localhost:3000

## Frontend
Open `Saarthi.html` in your browser (or use Live Server). The chat box will call `http://localhost:3000/api/chat`.

- If `OPENAI_API_KEY` is set, replies are generated by OpenAI.
- If not, a rule-based fallback provides supportive responses.

## Endpoints
- `POST /api/chat` → `{ reply: string }`
- `GET /health` → `{ ok: true }`

## Notes
- CORS is enabled for all origins to allow `file://` and Live Server during development.
- The assistant supports a crisis-safety branch with India helpline numbers.
